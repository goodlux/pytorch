Detailed Description

RAII guard that sets a certain default GPU index in its constructor, and changes it back to the device that was originally active upon destruction.

The index is always reset to the one that was active at the time of construction of the guard. Even if you set_index after construction, the destructor will still reset the index to the one that was active at construction time.
