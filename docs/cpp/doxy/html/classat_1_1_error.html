Detailed Description

The primary ATen error class. Provides a complete error message with source location information via what(), and a more concise message via what_without_backtrace(). Should primarily be used with the AT_ERROR macro.

NB: at::Error is handled specially by the default torch to suppress the backtrace, see torch/csrc/Exceptions.h
